# Form 425C

The Form 425C module (425C) is where the carrier prepares and tracks Form 425C records, keeping the supporting data organized and the filing history in one place.

## Overview
Form 425C is a compliance/reporting form the carrier is responsible for. The module gives it a dedicated home so the inputs are captured consistently and prior filings remain available for reference and audit.

## Key tasks
- **Prepare a 425C record** — capture the required fields for the form in one structured place.
- **Review prior filings** — keep a history of submitted 425C records so past periods can be referenced.
- **Cross-check supporting data** — confirm the underlying numbers tie back to the source records before finalizing.

## Tips & gotchas
- Treat 425C records as part of the compliance trail — keep history intact rather than overwriting prior entries.
- Verify the supporting figures against their source before finalizing a form, so a filing isn't built on stale data.

## FAQ
- **Where do prior 425C filings live?** In this module — past records are retained for reference and audit.
- **Can I edit a finalized form?** Preserve the filing history; corrections should be made as new/append entries rather than erasing the record.
