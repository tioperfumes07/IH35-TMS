═══ ITEM 4 — DEADLINE REMINDER CRON ═══

apps/backend/src/legal/matters-reminder.cron.ts:
  Runs daily at 08:00 Central Time
  Query: matter_deadlines where deadline_at within reminder_offset_days
  For each match:
    - Send email to reminder_recipients
    - Send PWA notification
    - Record send timestamp in reminder_sent_at array
  
  Special handling:
    - statute_of_limitations: also notifies Owner directly regardless
    - response deadlines: high-priority badge on UI
