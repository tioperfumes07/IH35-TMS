# Operator Runbooks — Index

Step-by-step procedures for the most common recurring office workflows. These
runbooks turn tribal knowledge into durable, repeatable processes.

| Runbook | Frequency | What it covers |
| --- | --- | --- |

| [IFTA Quarterly Filing](./IFTA-QUARTERLY-FILING.md) | Quarterly | IFTA-by-state report, miles/fuel validation, filing, payment, QBO JE. |

| [1099 Contractor Payroll](./1099-CONTRACTOR-PAYROLL.md) | Per cycle + year-end | Driver settlements, $600 threshold tracking, year-end 1099-NEC. |

| [Bank Reconciliation](./BANK-RECONCILIATION.md) | Daily + monthly | Plaid sync, categorization, matching, period reconcile + lock. |
| [Fuel Card Import](./FUEL-CARD-IMPORT.md) | Weekly | Love's/WEX imports, jurisdiction validation, QBO bill, allocation. |
| [Driver Onboarding Checklist](./DRIVER-ONBOARDING-CHECKLIST.md) | Per driver | Pre-hire screens, DQF, system setup, training, first load. |
| [Unit Acquisition Workflow](./UNIT-ACQUISITION-WORKFLOW.md) | Per unit | Purchase, title/DOT, insurance, Samsara, PM schedule, assignment. |

## Conventions

- Each runbook lists **Owner**, **Frequency**, **Systems**, ordered **Steps**, a
  **Verification** checklist, and an **Escalation** path.
- Runbooks are also surfaced in-app under **Help → Runbooks**.
