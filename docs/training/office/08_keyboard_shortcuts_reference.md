## Common questions
**Q: Shortcut is not working. Why?**  
A: Usually focus is outside the app, or browser-level shortcuts are overriding.

**Q: Is quick search better than filter in every case?**  
A: Use quick search for known IDs/names, and filters for queue triage.

**Q: Can I use shortcuts inside all modal fields?**  
A: Most navigation works, but text inputs can change behavior.

## What to do if something breaks
- Click app area, then retry shortcut.
- Use menu path if browser shortcut conflicts.
- Clear filters and retry quick search if results seem incomplete.
- Escalate repeat focus/key issues with browser version details.

## Shortcut list
- `Ctrl/Cmd + K`: open global search
- `Tab`: next field
- `Shift + Tab`: previous field
- `Esc`: close active modal/drawer
- `Enter`: confirm focused action
- `Arrow keys`: navigate list/options

## Tab navigation tips
- Move top-to-bottom with `Tab`
- Use `Shift + Tab` for quick corrections
- Recheck required fields before final submit

## Quick search tips
- Search by load number first
- Search by driver/customer when ID is unknown
- Clear filters if search seems incomplete
