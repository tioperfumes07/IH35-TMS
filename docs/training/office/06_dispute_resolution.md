# Dispute Resolution

## What you'll learn
- How to handle a driver dispute on a line item
- How office review should be documented
- Difference between adjustment and full resolution
- How to close the dispute with clear communication

## Before you start
- You need dispute and settlement access
- Gather supporting records first
- Confirm who can approve final resolution

## Common questions
**Q: When should we use adjustment?**  
A: Use it when only part of a line item is wrong.

**Q: When should we use full resolution?**  
A: Use it when the full line item must be reversed or replaced.

**Q: Can we close a dispute without notifying driver?**  
A: No, close-loop communication is required for clarity and trust.

## What to do if something breaks
- If record does not save, copy notes locally, then retry.
- If evidence is missing, pause and request missing records.
- If totals become inconsistent after changes, escalate before final close.
- Report repeated system failures with dispute ID and screenshots.
