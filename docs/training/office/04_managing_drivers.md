# Managing Drivers

## What you'll learn
- How to add a new driver profile
- Which required fields must be complete before assignment
- How to capture Mexican driver fields when applicable
- How to manage status changes

## Before you start
- You need permission to create/edit driver records
- Have CDL and compliance documents ready
- Confirm operating company assignment

## Step-by-step
1. Open Drivers and click **+ Create Driver**.
   [screenshot: Drivers header with create action]
2. Fill required fields:
   - CDL details
   - Expirations
   - Pay basis
   - Operating company
   [screenshot: Required fields in driver form]
3. If applicable, complete Mexican fields:
   - Visa
   - INE
   - CURP
   [screenshot: Mexican driver details section]
4. Save and verify the profile appears in All Drivers.
   [screenshot: New driver in All Drivers table]
5. Set status as needed:
   - Active
   - Inactive
   - On Leave
   [screenshot: Driver status control]
6. Recheck expiration dates before assignment.
   [screenshot: Compliance expiration summary]

## Common questions
**Q: Can a driver be assigned with near-expiry documents?**  
A: Follow compliance policy; do not bypass required safety checks.

**Q: On Leave vs Inactive?**  
A: On Leave for expected return. Inactive for non-active roster status.

**Q: Who approves pay basis changes?**  
A: Follow company approval policy before changing compensation settings.

## What to do if something breaks
- If save fails, recheck required fields and date formats.
- If status change does not persist, refresh and verify permissions.
- If driver does not appear in assignment lists, verify status and company.
- Escalate repeated issues with driver ID and screenshots.
