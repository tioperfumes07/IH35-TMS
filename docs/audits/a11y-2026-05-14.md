# Accessibility audit — 2026-05-14

## Scope

Automated scan with `@axe-core/cli` against production URLs (WCAG 2.0 Level A and AA rules).

## Command

```bash
npm run test:a11y
```

Equivalent:

```bash
axe https://app.ih35dispatch.com/dashboard \
  https://app.ih35dispatch.com/loads \
  https://app.ih35dispatch.com/settlements \
  https://app.ih35dispatch.com/banking \
  https://app.ih35dispatch.com/drivers \
  https://app.ih35dispatch.com/customers \
  --tags wcag2a,wcag2aa --exit
```
